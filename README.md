# Cinematic Portfolio

A dark, cinematic personal portfolio for a Software Engineer specializing in Cloud, DevOps, and Backend development.

## Stack

- Next.js App Router
- Framer Motion for section and card animations
- GSAP ScrollTrigger for scroll-linked transitions and parallax
- Three.js for the hero particle background
- Tailwind CSS for styling

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open `http://localhost:3000` in your browser.
